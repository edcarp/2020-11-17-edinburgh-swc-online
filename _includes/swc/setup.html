Software requirements for the Bash Shell (Tuesday) and Git (Thursday) lessons

The sections below (The Bash Shell, Git and Text Editor) tell you what software you'll need to have (or install) on your own computer in order to do the Bash Shell and Git lessons on Tuesday and Thursday.

As a possible alternative, if you are already familiar with either of the following University of Edinburgh services then you can use them for the Bash Shell and Git lessons.

  • Eddie, The University of Edinburgh Compute Cluster.
  • The School of Physics & Astronomy's SSH Login Server (staff.ph.ed.ac.uk).
